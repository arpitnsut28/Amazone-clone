# Amazon Clone

A simple Amazon homepage clone built using HTML and CSS.

## 🚀 Features

- Amazon-inspired homepage design
- Responsive navigation bar
- Hero section
- Product category boxes
- Footer section
- Clean and organized layout using CSS

## 🛠️ Technologies Used

- HTML5
- CSS3

## 📸 Preview

This project is a front-end clone created for learning and practicing web development fundamentals.

## 🎯 What I Learned

- Structuring web pages with HTML
- Styling layouts with CSS
- Using Flexbox for positioning elements
- Creating responsive and visually appealing UI components
- Organizing project files and assets
